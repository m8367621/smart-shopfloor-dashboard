ℹ️ About Smart Shopfloor Environment Monitoring System

A real-time industrial environment monitoring system developed using ESP32 and IoT technology to improve worker safety, monitor environmental conditions, and provide intelligent alerts.

📌 Project Overview

The Smart Shopfloor Environment Monitoring System continuously monitors dust concentration, sound level, temperature, humidity, and ambient light inside industrial shopfloors. The system displays live data on a web dashboard and automatically activates warning indications through a tower lamp whenever predefined safety thresholds are exceeded.
